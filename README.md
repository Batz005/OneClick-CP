# OneClick-CP

**OneClick-CP** is your ultimate companion for Competitive Programming (CP) and Data Structures & Algorithms (DSA) in VS Code. It streamlines your workflow with one-click file resets, solution exporting, and a comprehensive snippet library for C++, Java, Python, and JavaScript.

### üöÄ Multi-Language Support
Seamlessly switch between **C++**, **Java**, **Python**, and **JavaScript**. The extension automatically detects the language based on your active file or workspace structure.

### üîÑ One-Click Reset
Instantly reset your `main` file and input/output files to a clean state using your preferred template. No more manual clearing of files between problems!

Access an exhaustive collection of snippets for algorithms and data structures across all supported languages.

- **Data Structures**: Segment Trees, Tries, Union Find, Fenwick Trees, Linked Lists, Trees.

- **Snippet Explorer**: Browse and insert snippets with a preview on hover.
- **Layout Manager**: Arrange your editor layout (Code, Input, Output) with a single click.

### üíæ Custom Templates & Snippets
- **Save Templates**: Create custom templates from your current open files.
- **Create Snippets**: Turn any selected code into a reusable snippet directly from the sidebar.

Export your current solution to a dedicated folder with a custom name, keeping your workspace clean and organized.

## Usage

1.  **Open the Sidebar**: Click on the OneClick-CP icon in the Activity Bar.
2.  **Select Language**: The extension auto-detects the language, or you can select it manually in the snippet explorer.
3.  **Reset**: Click "Reset Files" to prepare for a new problem.
4.  **Code**: Write your solution. Use snippets by typing their prefix or clicking in the sidebar.
5.  **Test**: Run your code against `input.txt`.
6.  **Export**: Click "Export Solution" to save your work.

## Extension Settings

This extension contributes the following settings:

*   `oneclick-cp.templates`: Configure default content for `main` files, `input.txt`, and `output.txt` for different templates.
